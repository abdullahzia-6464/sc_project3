All scripts below runs on localhost by default. To change ip or ports please update "src/config.py".
Please run shell and python files in a separate terminal window to see logs.

1- Install necessary packages using:

	pip3 install -r requirements.txt 

2- Run ground control:

	./run_ground_control.sh

3- Run satellites:

	./run_satellites.sh

3- Run ship:

	./run_ship.sh

4- Run visualise.py to see all nodes and package transmission (if running on pis, port forwarding needs to be configured):

	python3 visualise.py

* To stop a satellite:
	
	./stop_port.sh <port>

* To stop all satellites:

	./stop_satellites.sh
